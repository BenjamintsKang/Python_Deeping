[ ]  **4. 회고를 잘 작성했나요?**
회고부분 뿐만 아니라 중간중간 주석을 통해 오류를 해결하면서 깨달은 점을 작성해놓으셨고, 회고를 통해 전체적인 후기에 대해 작성해 놓으셨다. 
        
[ ]  **5. 코드가 간결하고 효율적인가요?**
